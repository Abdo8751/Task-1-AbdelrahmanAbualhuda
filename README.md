## 🗂️ Dataset Overview

- **Source:** `city_temperature.csv`
- **Size:** ~2.9 million daily records
- **Columns:** Region, Country, State, City, Month, Day, Year, AvgTemperature (°F)
- **Coverage:** 321 cities · 125 countries · 7 world regions · 1995–2020

---

## 🔍 Key Findings

- Africa and the Middle East are the hottest regions (~26–27°C avg)
- Europe has the widest seasonal temperature swings
- A gradual global warming trend is visible across 1995–2020
- ~79K rows used `-99` as a missing temperature placeholder and were removed
